# Digital Apothecary

A web application for tracking and managing herbal remedies, providing detailed information about various herbs, and user intakes.

## Features

- User authentication for secure access
- Dashboard for an overview of herb management
- Individual herb detail pages 
- Intake tracker for recorded herbal usages
